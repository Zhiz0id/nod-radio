Android приложение Радио Национально-Освободительного Движения.

Эта страничка приложения НОД для Android, которое вы можете скачать прямо тут: https://github.com/Zhiz0id/nod-radio/blob/gh-pages/release/NOD-Radio.apk

Функционал

  • Прослушивание эфира http://radio.rusnod.ru

Поддержка и способ связи

Если нашли ошибки, неполадки или просто есть предложения по улучшению функционала, то используйте, пожалуйста, систему решения проблем от Github https://github.com/Zhiz0id/nod-radio/issues

Связаться с разработчиком можно по почте nod@you-ra.info .
